# CoinTrader
OKX平台自动化交易框架，策略交易框架

1、支持无限开多币种交易，只要你的主机性能够强<br/>
2、支持可定制策略，支持动态编译策略文件，内置策略编译器。 支持完整的C#语法<br/>
3、支持多策略并存<br/>
4、支持策略分组<br/>
5、支持合约现货同时运行<br/>
6、开放源代码， 没有API泄漏风险
